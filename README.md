# SEAWeb
一个简单的api模板网页

## <a href="https://molanp.github.io/SEAWeb_docs/" target="_blank">帮助文档</a>

## 简单配置

- 要求：PHP>=5.0 Apache
- 1. 上传文件至服务器
- 2. 访问`http://{your_website_url}`进行初次配置
- 3. Run!
